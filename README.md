## Projects

### [Batch Transaction Processing Engine](https://github.com/adithya-pillai/batch-txn-engine) &nbsp;·&nbsp; `C`
High-throughput batch processor with configurable concurrency, rollback semantics, and audit logging. Designed to demonstrate systems-level correctness under failure conditions.

### [Round-Robin Task Scheduler](https://github.com/adithya-pillai/rr-scheduler) &nbsp;·&nbsp; `C`
Preemptive scheduler with configurable time quanta, priority queues, and starvation guards. Built as a study of OS scheduling primitives in user space.

### [TriageAI](https://github.com/adithya-pillai/triageai) &nbsp;·&nbsp; `Python · LLM`
AI-assisted support ticket triage system with multi-label classification and confidence-gated human escalation. Reduces mean time to first response on high-volume queues.

### [Alert Notification Service](https://github.com/adithya-pillai/alert-notification-service) &nbsp;·&nbsp; `Python · AWS`
Modular multi-channel alert dispatch system (email, SMS, webhook) with retry logic, delivery tracking, and operator dashboards.
